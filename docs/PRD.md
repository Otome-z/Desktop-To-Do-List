# 产品需求文档
## 功能名称
桌面代表事项处理工具（Vue3 + TypeScript + Electron）

## 功能概述
- 启动后自动打开主窗口，使用卡片式网格展示任务；空态提示用户创建任务（简洁文字 + 新建按钮）。
- 支持创建任务/提醒、编辑、删除，任务仅包含标题、描述、提醒时间、完成状态，数据保存在 IndexedDB 中。
- 支持提醒（固定时间、倒计时、重复）和系统通知；提醒到达时弹出应用内浮层提示，同时发送通知，点击通知聚焦主窗口并展开相关任务。
- 托盘常驻，提供打开主窗口、新建任务和退出项；窗口可最小化至托盘，托盘图标显示状态指标。
- 通知内提供「完成」与「稍后提醒」按钮，提醒完成后同步任务状态。

## 用户旅程
1. 启动应用时自动打开主窗口并展示任务卡片列表；若无任务则显示空状态提示并鼓励新建。
2. 用户点击「新建任务」后输入标题、描述、提醒时间，任务立即展示在卡片网格中；点击卡片可在右侧展开详情与操作按钮。
3. 至少支持提醒在定时、倒计时或重复规则下触发，任务到点时在应用中弹出浮层并推送系统通知，用户可直接在通知中完成或 snooze。
4. 通知点击后聚焦主窗口且展开该任务的详情面板，用户可查看或继续编辑。
5. 窗口可最小化至托盘，并通过托盘菜单快速打开、新建或退出。

## 关键验收标准
- 提醒在设定时间 ±5 秒内弹出应用浮层与系统通知。
- 用户能在最多 3 步内创建新任务，并看到其出现在卡片网格中。
- 托盘操作（打开、最小化、退出）与通知点击能正确聚焦主窗口并展开展示内容。
- IndexedDB 与提醒状态保持一致，退出/重启不丢失任务或设置。
- 仅支持 Windows 平台，自动启动并保持托盘常驻。
